This is a font awesome cheatsheet. I made this because I found myself misclicking alot and not finding what I want on the original cheatsheet.

With this fontawesome cheatsheet you can

  • search by name and class
  • search by unicode
  • search by alias
  • intant copy to code to clipboard when you click on the icon html
  • instant copy unicode to clipboard
  • sort the table anyway you like

Use the search field to start searching for the icon you need.

If you click on the HTML or the unicode its under your CTRL V
